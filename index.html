À propos de Midzo

Midzo est le premier site d'achat de billets d'avion assisté par une IA conversationnelle. Notre agent de voyage virtuel vous accompagne comme un vrai agent humain — en posant les bonnes questions, en comprenant vos préférences, et en trouvant les meilleures offres.

Notre technologie

Nous utilisons des APIs de recherche de vols en temps réel couplées à une IA. Les clés API sont sécurisées côté serveur.

Données collectées

Midzo collecte uniquement les informations saisies dans le chat : ville de départ, destination, dates, passagers, budget optionnel.

Utilisation

Ces données sont transmises à nos partenaires API (Kiwi Tequila, Aviasales) pour générer les résultats en temps réel. Elles ne sont pas stockées au-delà de la session ni revendues.

Sécurité

Les clés API sont gérées côté serveur uniquement, jamais exposées dans le code client.
